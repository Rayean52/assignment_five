Question-1: What are the different ways to select an element in the DOM?
There are many different ways to select an element like getElementById, getElementByClassName, querySelectorAll, querySelector, getElementByTagName.

Question-3: What is event delegation in the DOM?
Event delegation is a technique in JavaScript that allows you to handle events efficiently by using a single event listener on a parent element to manage events on multiple child elements.
Question-4: What is event bubbling in the DOM?
Event bubbling is a phase in the event propagation process where an event triggered on a child element moves up through its parent elements, all the way to the root of the document.
